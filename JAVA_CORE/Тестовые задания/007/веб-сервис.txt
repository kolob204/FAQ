web-сервис  прогноза погоды


Требования
JAVA
детали - на устмотрение разработчика  (работодатель даёт полную свободу)

Постановка задачи
сервис отображает информацию о погоде с различных
погодных сервисов.
При входе в интерфейс пользователь может выбрать город и 
погодный сервис.

Функционал
1. минимум Два погодных сервиса
   минимум два показателя погоды (например  температура и влажность)

2. Сохранять настройки пользователя при выходе из браузера
   возобновлять при открытии

3. Кэшировать результат запроса пользователя на 5 минут (или другое время)
