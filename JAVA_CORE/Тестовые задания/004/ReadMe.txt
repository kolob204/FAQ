Задание 1

Есть веб-страница с таблицей. В таблице записи — это информация о клиентах.

При клике на одну строку (на кнопку на строке), переходим на развёрнутую информацию о клиенте. Это тоже таблица. В ней есть поле для фотографии. Сделать возможность загружать фотографии и изменять. На первой странице в таблице есть ссылки на фотографии. На второй странице (с подробной информацией) подтягивается изображение. Расположение файлов фотографий на файловой системе. Привязку данных о клиенте с местом на файловой системе хранить в БД PostgreSQL (Oracle)

Задание 2.

Есть таблица на SQL сервере с информацией о должниках. Есть таблица с шаблонами писем.

Нужно вытащить информацию о долниках и вставить их в шаблон, после чего сохранить в doc-файл.

Сделать интерфейс в котором видно инфорацию о должниках. Кнопку для формирования файла с выбором шаблона. После нажатия на кнопку «Сгенерировать файл» появляется ссылка для скачивания этого файла.

Использовать технологии:

Spring Framework (Spring MVC, Spring JDBC)

Интерфейс: JSP-страница с HTML, CSS, JavaScript

Проект должен быть веб-приложением (Java Web application), работающим на сервере Apache Tomca
